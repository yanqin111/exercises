该spark程序针对第二题：

原题目内容如下：

  有一个这样的文件，大小为2T, 里面的内容记录着一段时间内一批用户在腾讯游戏平台上玩各种游戏(如王者荣耀/QQ飞车)的记录。如以下
|对战开始时间|对战结束时间|玩家QQ号|其余对局玩家|使用角色（ID）|表现分|胜负（1为胜，0为负）|游戏ID|
|1548654558|1548655574|1234666|173123778,1123123,21123312,223211,1111,134248,999871,183712391238,1311331|6|55|1|300|
|1548655558|1548656574|1234666|173123778,1123123,21123312,111211,1111,134248,999871,183712391238,1311331|333|60.1|0|200|
|1548656590|1548657574|1234666|173123778,1123123,21123312,333211,1111,134248,999871,183712391238,1311331|1221212|79.2|1|801|
期望从该文本文件中，找出某个玩家擅长于使用哪些英雄/与哪些队友开黑胜率高/擅长于玩哪个游戏? 
提示：
 考虑效率/使用体验


解题思路如下：

    1.考虑到数据量比较大，所以使用大数据技术spark进行处理，一次性将所有玩家的数据计算完成，然后倒入到数据库中，
       对于具体的玩家，直接查询计算完成后的结果

    2.对于擅长的英雄角色，思路是：
        找出玩家最常用的游戏ID的游戏数据(为了规避玩家用小号导致表现分过高问题)，
        计算每个角色的平均表现分，表现分最高的则为最擅长的英雄

    3.对于与那些队友开黑胜率较高，思路是：
        计算玩家与常开黑队友(暂定为对局5次以上)的参考值降序排列，即是合适队友的排序，参考值分为：胜率参考值，对局时长参考值，表现分参考值

        a).胜率参考值：胜率参考值最大暂定60，计算玩家与每个常开黑队友对局的胜率，并与胜率最大值值相乘，得到胜率参考值

        b).对局时长参考值：对局时间参考值最大暂定15；算出玩家与每个队友每局游戏时长参考值，并求其平均值为最后的时长参考值
                        计算方式为：取全部数据的最短得胜对应15分，最长败局时间对应0分，并将每局时长得分映射到0-15区间

        c).表现分参考值：表现分参考值最大暂定25；算出玩家与常开黑队友每局的表现分参考值，并求平局值作为最后的表现分参考值
                       计算方式为：取全部数据中表现分最高的分数对应25分，并将表现分映射到0-25区间

    3.擅长玩哪个游戏：思路是：
        计算玩家Id所有对局胜率参考值和时长参考值和玩家的总游戏时长参考值，他们之和为最总参考值，降序排列即为擅长游戏的排序

        a).胜率产考值：最大值暂定60；算出玩家胜率并映射到0-60区间

        b).游戏总是长参考值：最大值暂定40且总时长100小时为满分，并将总时长映射到0-40分的区间
